Features

MULTIPLE REGISTRIES

Create multiple registries. Configure registries, policies and plugins to fit your workflow

MULTIPLE UPSTREAMS

Connect multiple upstream registries, both private and public (like registry.npmjs.org)

PRIVATE PACKAGES

Push packages into a private registry and share them with your team without having to make them public

PATCH PUBLIC PACKAGES

Push own version of a public package with the same name and version into your private registry

PUSH

Push a package from one registry to an upstream registry

PULL

Pull package from one or more upstream registries

CLONE REGISTRY

Copy an existing private registry, and all its contents, into a newly created one with a simple click

CONSOLE

Use the web console to visualize and manage your registries, upstreams and packages

CLI

Live on the command line? All important actions on registries and packages are available in the Bytesafe CLI

Enterprise features
