When performing nearest neighbour matching in R, is it possible to view the identity of which cases matched with which controls?

r rstudio nearest-neighbor propensity-score-matching

I'm first trying this out in R Studio with a small practice dataset found here (584 obs, 5 variables) (https://scholarworks.umass.edu/cgi/viewcontent.cgi?article=1330&context=pare)

Using this code I can use nearest neighbour matching to find the mean difference in matched cases and controls (1:1) where stw is my grouping variable and tot, min, and dis are the matching variables:

`m.out = matchit(stw ~ tot + min + dis, data = mydata, method = "nearest", ratio = 1)'

what i want to to know is how can i view which cases have matched with which controls (i.e. shows me the exact identity)?

I would also like to do this where I match 5 nearest neighbours (i.e. ratio = 5) and also view those exact identities. is there additional code needed for this?
